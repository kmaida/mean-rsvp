# reStart-mean

Boilerplate MEAN stack application with simple authentication via Oauth.

* Node
* Express
* MongoDB
* Gulp
* AngularJS (with [reStart-angular](https://github.com/kmaida/reStart-angular))
* [Satellizer](https://github.com/sahat/satellizer)
* FontAwesome
* Bootstrap 3
