# mean-rsvp

MEAN stack RSVP application

* MongoDB / Express / AngularJS / Node (with [reStart-mean](https://github.com/kmaida/reStart-mean))
* Token-based authentication (with Oauth via [Satellizer](https://github.com/sahat/satellizer))
* FontAwesome
* Bootstrap 3
* Gulp
